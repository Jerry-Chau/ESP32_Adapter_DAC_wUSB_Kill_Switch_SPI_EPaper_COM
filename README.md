# ESP32_Adapter_DAC_wUSB_Kill_Switch_SPI_EPaper_COM
These two boards allows an ESP32 to perform DAC from 0-10V using I2C and control to an E-Paper display. The main controller performs DAC using the MCP4725A0T-E/CH chip and non-inverting OpAmp.
